Practica 1

Ejercicio 1

Cree un array con una longitud introducida por el usuario y rellénelo con números aleatorios del 0-10 (usa Math.floor y Math.random). Se debe comprobar que la longitud es un número. (1 pto)

Ejercicio 2

Calcule su media aritmética (Si quieres usa Array.reduce pero no es obligatorio). (1 pto)


Ejercicio 3

Devuelve un nuevo array con los mayores o iguales a 5 (usa array.filter). (1 pto)


Ejercicio 4

Del array original devuelve un nuevo array con la multiplicación de todos los elementos por dos (usa array.Map). (1 pto)


Practica 2

Ejercicio 1

Cree un array de personas (al menos 5). Puede insertarlas manualmente sin usar prompt ni DOM. Las personas tendrán un campo id (el cual será único), nombre, apellidos, y edad. Utilice notación JSON. (0,5 pto)

Ejercicio 2

Ordene el array original de mayor a menor edad. (1 pto)

Ejercicio 3

Crea una función que dados el array y un id, elimine esa persona del array. Si el id no existe deberá mostrar un alert con un mensaje indicándolo. Pon un fragmento de código en el que se vea que funciona. (1,5 pto)

Ejercicio 4

Crea una copia del array anterior en un nuevo array. (1 pto)

Ejercicio 5

Concatena ambos arrays y elimina los elementos repetidos. (2 pto)
